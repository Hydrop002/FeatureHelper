# FeatureHelper

使用命令生成洞穴、峡谷、结构和地物。

![img](https://github.com/Hydrop002/FeatureHelper/blob/master/img/main.png)

已实现：
* 结构组件生成
* 结构生成
* 结构单步生成
* 结构NBT支持
* 结构碰撞盒渲染
* 特征生成
* 洞穴生成
* 洞穴单步生成

待实现：
* 峡谷、下界洞穴生成
* 峡谷、下界洞穴单步生成

---

搭建项目环境之前，你需要：
1. 解压eclipse.zip，如果你使用eclipse开发的话
2. 在gradle.properties中配置你的gradle-user-home
